# Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Contact:
Email us at tech-incridea@nmamit.in

# What to include:

A detailed description of the vulnerability

# Steps to reproduce the issue

Any relevant logs, screenshots, or proof-of-concept code

Response timeline:
We will acknowledge receipt of your report as soon as possible and provide updates throughout the investigation process.

Disclosure policy:
Please do not publicly disclose the vulnerability until it has been reviewed and resolved by our team.

We appreciate your efforts to help keep this project and its users secure.
